How to Laser Cut

navigate using left and right arrows or swiping left or right on your smart phone

Create Geometry

  • Vector lines must be used if desired outcome is to cut materials
    (see example)
  • Vector lines can also be used fo engraving linework
    (see Example)
  • Greyscale Images can be used to 'raster' and image onto material
    (see example)
    •

    Organize Geometry For Material

  • Draw outline of material, Length x Width
    (see example)
  • Organize geometry within the materials boundry
    (see example)
    •

    Import/Copy into Laser Template

  • Save original file
    (see example)
  • Import or copy past original file into laser template file
    (see example)
    •

    Specify Properties

  • Place Cut lines on the same layer, verify their properties match a sample line from template file
    (see example)
  • Place engrave lines on the same layer, verify their properties match a sample line from template file
    (see example)
  • Place greyscale images on their own layer, verify the layer color is 'black'
    (see example)
    •

    Print Properties

  • Click File -> Print
    (see example)
    • Use the settings sheet to specify the following for material being used

    (example of seeting sheet)
  • Specify the properties (power, speed, PPI) for each line color.
    (see example)
  • Specify if the color should be a vector cut/engrave, a raster for greyscale images or skipped
    (see example)
  • Specify whether to use one or two laser tubes(if more than one is available)
    (see example)
    •

    Print Properties Continued

  • Select 'window' from 'view and output scale
    (see example)
  • Drag a window around print area (should be a rectangle measuring 32"x18")
    (see example)
  • Verify the print preview is correct (print preview is a representation of the laser bed)
    (see example)
    •

    Open UCP Software

  • Click on the desktop icon, Menu Icon or use alt+tab to open the UCP software
    (see example)
  • Ensure the file appears correctly in the Preview
    (see example)
    •

    Turn Laser On

  • Swipe Harvard ID Card in Card Reader to turn Laser on
    (see example, 3d model)
    •

    Materials Placement

  • Place Material in the laser in the same orientation as drawn.
    (see example, 3d model)
    •

    Focus Lens

  • On laser control panel, Press the Focus button
    (see example, 3d model)
  • Place the focus stick on material and use arrows to raise and lower bed
    until carriage sit appropriately on focus stick.
    (see example, 3d model)
  • Press return key on control panel.
    (see example, 3d model)
    •

    Press Play

  • Open UCP software and press the green play button.
    (see example, 3d model)
  • Remove material and throw offcuts/scraps in trash.
    (see example, 3d model)
    •
